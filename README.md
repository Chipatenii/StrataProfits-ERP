# StrataForge Business Suite

## Overview

StrataForge Business Suite is a comprehensive, remote-first operational OS designed to bridge the gap between sales, project management, and finance. It empowers agencies and service-based businesses to seamlessly manage clients, deals, projects, tasks, time-tracking, and financials all in one place.

The platform provides distinct, role-based experiences:

- **Admin Dashboard**: For high-level oversight of the entire business lifecycle—from creating and assigning tasks to monitoring team workload, managing the CRM pipeline, and reviewing detailed financial/monthly reports.
- **Team Portal**: For core team members to view their assigned tasks, track active work using the built-in timer, and collaborate efficiently.
- **Virtual Assistant (VA) Portal**: A tailored workspace for VAs with restricted access to specific tasks, SOPs, and financial data relevant to their role.

With features like automated invoice generation, quote proposals, approval workflows, and Standard Operating Procedures (SOPs), StrataForge acts as the central hub for running a modern remote business.

## Tech Stack

- **Frontend:** Next.js 16, TypeScript
- **UI:** Tailwind CSS, Shadcn/UI
- **Backend & Auth:** Supabase (database, authentication, and API)
